5.1 碳市场经济学原理

5.1.1 外部性与市场失灵

碳排放是典型的负外部性问题。企业排放温室气体造成的环境损害未计入私人成本，导致市场失灵。

MSC = MPC + MEC（社会边际成本 = 私人边际成本 + 边际外部成本）

由于MPC < MSC，导致市场均衡产量大于社会最优产量。

5.1.2 庇古税与科斯定理

  • 庇古税：对污染排放征税，使企业承担边际外部成本 t* = MEC(Q*)
  • 科斯定理：如果交易成本为零且产权明确，市场可通过谈判解决外部性问题

5.1.3 总量控制与交易机制

碳排放权交易系统（ETS）通过设定排放总量上限并允许配额交易，实现减排成本的社会最优分配。

均衡条件: MAC = P = MD（边际减排成本 = 碳价格 = 边际损害）

5.1.4 边际减排成本曲线（MACC）

MACC描述了在不同减排水平下的边际成本，用于政策制定、碳价预测和企业决策。

MAC_i = (Cost_i - Benefit_i) / ΔE_i（边际减排成本 = 单位减排成本 - 协同收益）

Allocation = Benchmark × Output × AF

基准线法奖励高效企业，促进技术进步，是未来发展方向。

  • 市场预期：未来政策走向、行业扩容进度

5.3 CCER项目开发

5.3.1 CCER概述

CCER（中国核证自愿减排量）是对我国境内可再生能源、甲烷回收、造林等减排项目产生的减排量进行核证，并在碳市场交易的活动。

5.3.2 方法学体系

可再生能源类：CMS-001（光伏）、CMS-002（风电）、CMS-003（水电）、CMS-004（生物质）、CMS-005（沼气）

造林碳汇类：AR-CM-001（乔木造林）、AR-CM-002（竹林）、AR-CM-003（红树林）

甲烷回收类：CMS-007（煤矿瓦斯）、CMS-008（垃圾填埋气）、CMS-017（畜禽粪便沼气）

5.3.3 项目开发流程

  1. 项目识别：评估是否符合方法学要求
  2. 方法学选择：选择适用的方法学
  3. 额外性论证：证明项目具有额外性
  4. 编制PDD：编写项目设计文件
  5. 审定：第三方机构审定
  6. 项目注册：在主管部门注册
  7. 监测与核查：按监测计划实施
  8. 减排量签发：申请签发CCER

5.3.4 额外性论证

额外性是指项目相对于基准线情景产生了额外的减排效果：

  • 技术额外性：项目采用的技术非常规
  • 投资额外性：项目在没有减排收益时不具财务可行性
  • 障碍额外性：存在实施该项目的障碍

5.3.5 减排量计算

ER_y = BE_y - PE_y - LE_y（减排量 = 基准线排放 - 项目排放 - 泄露排放）
📊 10MW分布式光伏CCER开发案例

项目概况：装机10MWp，年发电量1200万kWh，江苏地区

基准线排放：BE = 1200万kWh × 0.5810 tCO₂/MWh = 6972 tCO₂/年

项目排放：PE ≈ 0（光伏无直接排放）

年减排量：6972 tCO₂/年

25年累计减排：约17.5万吨CO₂

CCER收益（50元/吨）：约35万元/年

5.5 碳金融产品

5.5.1 碳期货

碳期货是碳排放权标准化合约，具有价格发现、套期保值、投机获利功能。

套保比率 = Cov(S, F) / Var(F)

其中：Cov(S,F)为现货与期货价格协方差；Var(F)为期货价格方差

5.5.2 碳期权

碳期权赋予持有者以约定价格买入（看涨）或卖出（看跌）碳排放权的权利。

C = S × N(d₁) - K × e^(-rT) × N(d₂)（Black-Scholes定价）

其中：C为期权价格；S为标的资产当前价格；K为执行价格；r为无风险利率；T为到期时间；σ为标的资产波动率；N(d)为标准正态分布累积分布函数

d₁ = [ln(S/K) + (r + σ²/2)T] / (σ√T)

其中：d₁为正态分布参数；其他变量同上

5.5.3 碳质押

企业以碳排放权作为质押物获取融资。

LTV = Loan / (Carbon_Price × Carbon_Volume × LGD)

其中：LTV为贷款价值比；Loan为贷款金额；Carbon_Price为碳价；Carbon_Volume为碳配额数量；LGD为违约损失率

5.5.4 碳债券

将碳资产收益与债券挂钩的创新融资工具。

债券收益 = 固定利息 + max(0, (P_market - P_base) × Quantity)

其中：P_market为市场价格；P_base为基准价格；Quantity为数量

5.6 碳市场风险管理

5.6.1 风险类型

风险类型描述影响
价格风险碳价波动导致资产价值变化直接影响盈亏
信用风险交易对手违约无法完成交易
流动性风险市场流动性不足无法以合理价格买卖
合规风险数据造假、报告违规处罚、声誉损失
政策风险政策变化影响碳市场配额规则、市场机制改变

5.6.2 风险量化

VaR: P(Loss > VaR) = 1 - α（在置信水平α下，最大可能损失）
ES = E[Loss | Loss > VaR]（超过VaR的平均损失）

5.6.3 合规管理要点

  • 数据真实准确是生命线
  • 按时完成报告报送和履约清缴
  • 保留完整的台账和证据链
  • 积极配合核查和监管

💹 Part A：碳期货与碳期权

5.A.1 碳金融产品概述

碳金融是将碳排放权作为金融资产的创新金融形态，包括碳现货、碳期货、碳期权、碳远期、碳互换等金融工具。

产品类型定义主要功能成熟市场
碳现货即时交割的碳配额满足履约、套利各碳市场
碳期货未来某时点交割的标准化合约价格发现、套期保值EU ETS
碳期权购买/出售碳配额的权利风险对冲、灵活保值EU ETS
碳远期双方约定未来交易的非标准合约定制化保值OTC市场
碳互换不同碳资产/货币的交换资产优化配置机构间

5.A.2 碳期货（Carbon Futures）

5.A.2.1 碳期货定义与机制

碳期货是买卖双方约定在未来某一特定时间、以特定价格交割一定数量碳配额（EUA）或CCER的标准化合约。

碳期货价格 = 现货价格 + 持有成本（storage cost + convenience yield）

5.A.2.2 EU ETS期货合约规格

参数EUA期货备注
合约乘数1 EUA = 1吨CO₂标准单位
报价单位欧元/吨CO₂小数点后两位
合约月份12月/3月/6月/9月按季度到期
交易时间工作日 08:00-18:00 CETEEX交易所
持仓限额净多头/空头各有限制防止操纵
交割方式现金交割或注册账户转让无实物交割

5.A.2.3 碳期货套期保值

企业通过碳期货锁定未来碳价，降低履约成本不确定性。

【空头套保（卖出期货）】
保值量 = 预计碳配额缺口 × β（相关系数）

套保效果：碳价下跌时，期货盈利弥补现货损失
套保成本：期货价格 - 现货价格 = 基差风险
📊 碳期货套期保值案例

背景：某发电企业，预计2026年履约需购买100万吨碳配额，当前碳价40欧元/吨。

套保方案：卖出100万吨EUA期货合约（锁定价格40欧元）。

情景现货市场期货市场综合成本
碳价下跌至30欧元购买成本：30×100万=3000万期货盈利：(40-30)×100万=1000万实际成本：4000万
碳价上涨至50欧元购买成本：50×100万=5000万期货亏损：(40-50)×100万=-1000万实际成本：4000万
碳价不变40欧元购买成本：40×100万=4000万期货盈利：0实际成本：4000万

结论：无论碳价如何变动，实际采购成本锁定在4000万欧元，消除了价格波动风险。

5.A.3 碳期权（Carbon Options）

5.A.3.1 碳期权定义

碳期权赋予持有者在特定日期或之前，按约定价格买入（看涨期权）或卖出（看跌期权）碳配额的权利。

类型英文定义买方权利卖方义务
看涨期权Call Option买入碳配额的权利有权按行权价买入必须卖出（被行权时）
看跌期权Put Option卖出碳配额的权利有权按行权价卖出必须买入（被行权时）

5.A.3.2 碳期权收益结构

【买入看涨期权（保护性买权）】
收益 = max(S_T - K, 0) - 权利金
其中：S_T=到期日碳价，K=行权价

特点：碳价上涨时获得上涨收益，碳价下跌时仅损失权利金
【买入看跌期权（保护性卖权）】
收益 = max(K - S_T, 0) - 权利金

特点：碳价下跌时获得保护，碳价上涨时仅损失权利金

5.A.3.3 碳期权定价模型（Black-Scholes）

【碳期权BS定价公式】
C = S₀ × N(d₁) - K × e^(-rT) × N(d₂)

d₁ = [ln(S₀/K) + (r + σ²/2)T] / (σ√T)
d₂ = d₁ - σ√T

其中：
C=看涨期权价格，S₀=当前碳价，K=行权价
r=无风险利率，T=到期时间(年)，σ=碳价波动率
N(·)=标准正态分布累积函数
📊 碳期权定价实例

参数设定：

  • 当前碳价S₀ = 50欧元/吨
  • 行权价K = 55欧元/吨（OTM看涨期权）
  • 到期时间T = 1年
  • 无风险利率r = 3%
  • 碳价年化波动率σ = 35%（EU ETS历史波动率25-50%）

计算过程：

d₁ = [ln(50/55) + (0.03 + 0.35²/2)×1] / (0.35×1) = [-0.095 + 0.091] / 0.35 = -0.011
d₂ = d₁ - 0.35 = -0.361
N(d₁) = N(-0.011) = 0.496
N(d₂) = N(-0.361) = 0.359

C = 50×0.496 - 55×e^(-0.03)×0.359 = 24.8 - 19.1 = 5.7欧元/吨

结果：该碳看涨期权理论价格约为5.7欧元/吨，约占碳价的11.4%。

碳金融是将碳排放权作为金融资产的创新金融形态，包括碳现货、碳期货、碳期权、碳远期、碳互换等金融工具。

产品类型定义主要功能成熟市场
碳现货即时交割的碳配额满足履约、套利各碳市场
碳期货未来某时点交割的标准化合约价格发现、套期保值EU ETS
碳期权购买/出售碳配额的权利风险对冲、灵活保值EU ETS
碳远期双方约定未来交易的非标准合约定制化保值OTC市场
碳互换不同碳资产/货币的交换资产优化配置机构间

5.A.2 碳期货（Carbon Futures）

5.A.2.1 碳期货定义与机制

碳期货是买卖双方约定在未来某一特定时间、以特定价格交割一定数量碳配额（EUA）或CCER的标准化合约。

碳期货价格 = 现货价格 + 持有成本（storage cost + convenience yield）

5.A.2.2 EU ETS期货合约规格

参数EUA期货备注
合约乘数1 EUA = 1吨CO₂标准单位
报价单位欧元/吨CO₂小数点后两位
合约月份12月/3月/6月/9月按季度到期
交易时间工作日 08:00-18:00 CETEEX交易所
持仓限额净多头/空头各有限制防止操纵
交割方式现金交割或注册账户转让无实物交割

5.A.2.3 碳期货套期保值

企业通过碳期货锁定未来碳价，降低履约成本不确定性。

【空头套保（卖出期货）】
保值量 = 预计碳配额缺口 × β（相关系数）

套保效果：碳价下跌时，期货盈利弥补现货损失
套保成本：期货价格 - 现货价格 = 基差风险
📊 碳期货套期保值案例

背景：某发电企业，预计2026年履约需购买100万吨碳配额，当前碳价40欧元/吨。

套保方案：卖出100万吨EUA期货合约（锁定价格40欧元）。

情景现货市场期货市场综合成本
碳价下跌至30欧元购买成本：30×100万=3000万期货盈利：(40-30)×100万=1000万实际成本：4000万
碳价上涨至50欧元购买成本：50×100万=5000万期货亏损：(40-50)×100万=-1000万实际成本：4000万
碳价不变40欧元购买成本：40×100万=4000万期货盈利：0实际成本：4000万

结论：无论碳价如何变动，实际采购成本锁定在4000万欧元，消除了价格波动风险。

5.A.3 碳期权（Carbon Options）

5.A.3.1 碳期权定义

碳期权赋予持有者在特定日期或之前，按约定价格买入（看涨期权）或卖出（看跌期权）碳配额的权利。

类型英文定义买方权利卖方义务
看涨期权Call Option买入碳配额的权利有权按行权价买入必须卖出（被行权时）
看跌期权Put Option卖出碳配额的权利有权按行权价卖出必须买入（被行权时）

5.A.3.2 碳期权收益结构

【买入看涨期权（保护性买权）】
收益 = max(S_T - K, 0) - 权利金
其中：S_T=到期日碳价，K=行权价

特点：碳价上涨时获得上涨收益，碳价下跌时仅损失权利金
【买入看跌期权（保护性卖权）】
收益 = max(K - S_T, 0) - 权利金

特点：碳价下跌时获得保护，碳价上涨时仅损失权利金

5.A.3.3 碳期权定价模型（Black-Scholes）

【碳期权BS定价公式】
C = S₀ × N(d₁) - K × e^(-rT) × N(d₂)

d₁ = [ln(S₀/K) + (r + σ²/2)T] / (σ√T)
d₂ = d₁ - σ√T

其中：
C=看涨期权价格，S₀=当前碳价，K=行权价
r=无风险利率，T=到期时间(年)，σ=碳价波动率
N(·)=标准正态分布累积函数
📊 碳期权定价实例

参数设定：

  • 当前碳价S₀ = 50欧元/吨
  • 行权价K = 55欧元/吨（OTM看涨期权）
  • 到期时间T = 1年
  • 无风险利率r = 3%
  • 碳价年化波动率σ = 35%（EU ETS历史波动率25-50%）

计算过程：

d₁ = [ln(50/55) + (0.03 + 0.35²/2)×1] / (0.35×1) = [-0.095 + 0.091] / 0.35 = -0.011
d₂ = d₁ - 0.35 = -0.361
N(d₁) = N(-0.011) = 0.496
N(d₂) = N(-0.361) = 0.359

C = 50×0.496 - 55×e^(-0.03)×0.359 = 24.8 - 19.1 = 5.7欧元/吨

结果：该碳看涨期权理论价格约为5.7欧元/吨，约占碳价的11.4%。
