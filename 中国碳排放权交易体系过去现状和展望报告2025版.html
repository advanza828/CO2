⚙️ 全国碳市场制度框架与关键设计

📜 法律与法规体系

  • 《碳排放权交易管理暂行条例》 (2024.5.1施行) — 最高行政法规
  • 生态环境部《管理办法》及配额分配方案、核算指南等配套文件
  • 明确监管主体、处罚力度（最高违法所得10倍罚款）
  • 全国统一注册登记系统（武汉）+ 交易系统（上海）

🎯 覆盖范围 & 总量设定

  • 行业：电力、钢铁、水泥、铝冶炼（2025扩围），未来将纳入石化、化工、造纸等
  • 温室气体：CO₂为主，铝冶炼新增CF₄、C₂F₆
  • 纳入门槛：年排放≥2.6万吨CO₂当量
  • 基于强度的总量设定（无绝对总量）→ 配额=基准值×实际产量，随产出浮动
  • 第一个履约周期年均配额~45亿吨 → 第二周期~51亿吨 → 第三周期~52亿吨 → 扩围后覆盖80亿吨

🎫 配额分配与MRV

  • ✅ 全部免费分配（未来将引入有偿拍卖）
  • 发电行业：基准线法（4类机组），两步分配法：预分配70% + 事后调整
  • 新纳入行业（钢铁/水泥/铝）：2024年配额等量分配，2025-2026基于强度控制，行业盈亏平衡
  • MRV体系强化：月度存证+“国家-省-市”三级联审，严打数据造假
  • 核查由政府购买服务，保证独立性

💱 交易与履约机制

  • 交易方式：挂牌协议、大宗协议、单向竞价（±10% / ±30%涨跌幅）
  • 仅限重点排放单位参与，目前现货市场（衍生品暂未开放）
  • 履约：从两年一次改为每年履约，第三个周期起设置配额结转限制
  • 灵活机制：燃气机组免履约缺口，燃煤机组缺口上限20%
  • 处罚升级：未清缴配额处以5-10倍市场均价罚款

🌿 全国碳市场体系双支柱

💨 强制碳市场 (ETS) 🌱 自愿减排市场 (CCER)
  • CCER抵销机制：≤ 5%配额清缴，2024年新CCER机制重启，首批签发948万吨
  • 已发布6个方法学：造林碳汇、海上风电、光热发电、低浓度瓦斯利用等
  • CCER与全国碳市场、地方试点联动，激励全社会减排
  • 2025年1月起旧CCER不再用于全国市场履约，新体系全面接轨
📈 市场表现 & 碳价趋势
📊 2021第一履约周期：40-60元/吨 📈 2022-2023：升至50-82元/吨 🎯 2024年底收盘97.49元/吨 🔮 2030年预测接近300元/吨（约42美元）

🏙️ 地方碳市场经验 → 全国市场创新

🔍 8大地方试点成果

  • 深圳：控排企业碳强度较2013年下降49%，增加值增长62%
  • 湖北、广东、上海等试点强度显著下降，累计减排千万吨级
  • 创新碳金融：碳质押、配额回购、碳保险等先行先试
  • 构建MRV能力、第三方核查管理体系，为全国市场铺路

🚀 全国市场特色设计

  • ✔️ 基于强度的总量 + 基准线法，适应碳达峰前经济波动
  • ✔️ “月度存证+三级联审”保障数据质量全球领先实践
  • ✔️ 配额结转规则（净卖出量×1.5+1万吨）提升流动性
  • ✔️ 循序渐进扩围：优先发电→2025纳入三大工业，涵盖非CO₂温室气体

📐 MRV 数据质量保障里程碑

  • 2022年起发电行业月度信息化存证，数据异常实时预警
  • 《条例》大幅提升数据造假处罚：最高违法所得10倍罚款
  • 核查机构不符合项同比减少35.7%，一次整改合格率升至92%
  • 碳排放连续监测系统(CEMS)试点推进，未来技术融合

🌏 国际接轨与自愿减排市场

  • CCER机制获得国际民航组织CORSIA认可，未来或衔接《巴黎协定》第6条
  • 新CCER注册登记机构（国家气候战略中心）＋交易机构（北京绿交所）
  • 截至2025年6月，23个项目完成登记，减排量超900万吨
  • 碳普惠机制探索：连接个人低碳行为与碳市场
